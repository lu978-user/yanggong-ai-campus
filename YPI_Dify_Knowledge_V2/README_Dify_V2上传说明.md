# README：Dify V2 上传说明

## V2 定位

V2 是 YPI 成长导师知识库的“学院覆盖版”，用于补齐信息工程学院之外的学院基础成长规划。V2 不做班主任助理、蓝桥杯、专转本等专题细化；专题库可留到 V3 或后续版本。

## 知识库名称建议

建议将 V2 新增文件上传到同一个 Dify 知识库：

```txt
YPI成长导师知识库_V1
```

也可以在完成 V2 后将知识库统一改名为：

```txt
YPI成长导师知识库
```

## 回答边界提醒

涉及政策、奖助、证书、竞赛报名、就业数据、升学政策时，AI 成长导师必须提醒以学校、学院、江苏省教育考试院、赛事或证书官方最新通知为准。V2 主知识库不把就业率、获奖数量、教师人数、企业合作数量、培训人数、取证率、签约率和宣传性表达作为确定事实输出。
