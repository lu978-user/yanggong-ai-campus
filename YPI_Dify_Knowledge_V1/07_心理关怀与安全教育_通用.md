# 心理关怀边界与校内资源

【适用对象】
感到学习压力、考试焦虑、人际困扰、情绪低落或需要心理支持的学生。

【适用目标】
情绪支持、压力调节、校内资源引导。

【核心事实】
整理资料提到校级心理资源包括大学生健康成长指导中心（维扬楼 5 楼）、咨询电话 0514-87433340，以及个体咨询室、团体辅导室、宣泄室、心理阅览室等功能室。

【成长建议】
本知识库只能提供情绪支持、压力调节建议和校内资源引导，不做心理诊断，不替代专业心理咨询。

【推荐行动】
1. 先把当前困扰写成 1 至 2 个具体问题。
2. 尝试规律作息、适度运动和减少熬夜。
3. 与可信任的同学、辅导员或家人沟通。
4. 需要时联系学校心理资源。

【回答边界】
如果出现持续强烈痛苦、自伤念头、伤害他人念头或紧急危险，应立即联系辅导员、学校心理中心、家长或当地紧急救助渠道。

【可信度】
A：校级心理资源来自整理资料；危机边界可直接作为回答规范。

# 学习压力与考试焦虑

【适用对象】
因课程、证书、竞赛、专转本或就业准备感到压力的学生。

【适用目标】
缓解焦虑、恢复学习节奏、制定可执行计划。

【核心事实】
学习压力和考试焦虑常与目标过多、任务不清、复习节奏失衡有关。

【成长建议】
建议把大目标拆成本周和今天能完成的小任务，先恢复行动感，再逐步提高任务难度。

【推荐行动】
1. 列出最急的 3 个任务。
2. 每天安排 2 至 3 个专注学习时段。
3. 用错题或复盘代替单纯焦虑。
4. 如果焦虑持续影响睡眠和生活，主动寻求辅导员或心理中心支持。

【回答边界】
不评价学生是否“心理有问题”，不做诊断；只提供一般性调节建议和资源引导。

【可信度】
C：属于心理支持和学习方法建议。

# 校园安全教育

【适用对象】
所有在校学生，尤其是新生、住宿生、参加实训和社会实践的学生。

【适用目标】
消防安全、网络安全、交通安全、实训安全和反诈防范。

【核心事实】
整理资料提到安全教育内容包括宿舍用电安全、灭火器使用、消防应急疏散、预防电信网络诈骗、个人信息保护、不乘坐非法运营车辆、实训操作安全规范等。

【成长建议】
安全问题要优先遵守学校规定和现场老师要求。涉及个人信息、转账、兼职、刷单、中奖、贷款等情形时，应提高警惕。

【推荐行动】
1. 宿舍不使用违规电器，不私拉乱接电线。
2. 遇到转账、验证码、兼职刷单信息先核实。
3. 实训前了解设备和操作规范。
4. 外出实践注意交通和人身安全。
5. 遇到紧急情况及时联系辅导员、保卫处或报警。

【回答边界】
具体安全制度、演练安排、实训规范以学校和现场管理要求为准。

【可信度】
B：安全主题来自整理资料；具体制度需以学校通知为准。
