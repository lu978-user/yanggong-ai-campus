# 大一：信息工程学院专业入门行动清单

【适用对象】
信息工程学院大一学生，计算机类相关方向。

【适用目标】
专业入门、建立学习习惯、发现兴趣方向。

【本月行动】
1. 选择一门编程语言开始系统学习。
2. 每周完成 2 次基础编程练习。
3. 了解蓝桥杯或计算机类基础竞赛。
4. 加入一个技术类社团或学习小组。
5. 关注学院发布的竞赛、证书、实践机会。

【回答边界】
这是成长建议，不是学校强制要求。具体课程、活动和竞赛安排以学院通知为准。

【可信度】
C：成长建议。

# 大二：信息工程学院方向深化行动清单

【适用对象】
信息工程学院大二学生。

【适用目标】
确定专业方向、准备竞赛证书、积累项目经历。

【本月行动】
1. 在云计算、软件、网络、人工智能、大数据、物联网等方向中选定主攻方向。
2. 整理目标方向需要的 5 项核心技能。
3. 启动一个完整项目并记录开发过程。
4. 选择一个证书或竞赛作为本学期重点。
5. 每周复盘一次项目、课程和证书进度。

【回答边界】
证书、竞赛和项目机会以学院及官方通知为准。

【可信度】
C：成长建议。

# 大三：实习就业或专转本行动清单

【适用对象】
信息工程学院大三学生、毕业班学生。

【适用目标】
就业冲刺、跟岗实习、毕业设计、专转本复习。

【本月行动】
1. 明确本学期主线是就业还是升学。
2. 就业型学生完善简历和项目说明。
3. 升学型学生制定每周复习计划。
4. 完成一次模拟面试或阶段测评。
5. 主动与辅导员、专业老师或就业指导老师沟通。

【回答边界】
招聘、实习、考试和毕业安排以学校、学院和官方平台最新通知为准。

【可信度】
C：成长建议。

# 专转本目标行动清单

【适用对象】
有专转本或继续升学目标的学生。

【适用目标】
明确政策、规划复习、持续执行。

【本月行动】
1. 查询江苏省教育考试院和学校最新政策。
2. 确认可报考专业类别和考试科目。
3. 制定英语和专业基础复习计划。
4. 每周完成一次错题整理。
5. 月末复盘复习时长和薄弱模块。

【回答边界】
招生院校、专业、科目、报名条件和政策以官方最新通知为准。

【可信度】
B：升学方向可参考，具体政策需官方确认。

# 就业目标行动清单

【适用对象】
准备实习、找工作或提升就业竞争力的学生。

【适用目标】
岗位认知、技能补齐、简历准备。

【本月行动】
1. 选择 1 至 2 个目标岗位方向。
2. 收集 5 条招聘信息并提取共性技能。
3. 优化一个项目作品或实训经历。
4. 完成一版简历并请老师或同学反馈。
5. 练习一次项目介绍和自我介绍。

【回答边界】
岗位、薪资、招聘计划和录用结果受市场和个人能力影响，不做承诺。

【可信度】
C：就业准备建议。

# 竞赛目标行动清单

【适用对象】
希望参加技能竞赛、蓝桥杯、创新创业竞赛或数学建模的学生。

【适用目标】
竞赛入门、训练节奏、团队准备。

【本月行动】
1. 选择一个最匹配专业方向的竞赛。
2. 阅读上一届规则或通知。
3. 建立每周训练计划。
4. 寻找队友或指导老师。
5. 记录训练成果和问题清单。

【回答边界】
报名时间、规则、名额和参赛要求以赛事官方和学院通知为准。

【可信度】
B：竞赛方向可参考，具体安排需官方确认。

# 考证目标行动清单

【适用对象】
准备软考、1+X、计算机等级、英语等级或办公应用证书的学生。

【适用目标】
证书选择、备考计划、能力验证。

【本月行动】
1. 明确证书服务于就业、升学还是技能提升。
2. 查询当次考试报名时间和要求。
3. 制定每周复习和练习安排。
4. 完成一套样题或真题练习。
5. 把证书知识应用到项目或课程任务中。

【回答边界】
报名、考试、费用、大纲和通过标准以官方平台和学校通知为准。

【可信度】
B：证书方向来自整理资料，具体考试安排需官方确认。

# 综合素质目标行动清单

【适用对象】
希望提升学生工作、志愿服务、社团和综合素质的学生。

【适用目标】
校园参与、评优评先、沟通协作、简历经历。

【本月行动】
1. 选择一个学生组织、社团或志愿服务方向。
2. 关注招募和活动通知。
3. 记录自己的职责、投入时间和成果。
4. 参加一次活动复盘或总结。
5. 将经历整理成个人成长材料。

【回答边界】
招募条件、活动安排和评优认定以学校和学院通知为准。

【可信度】
C：成长建议。

# 暂无明确目标学生行动清单

【适用对象】
还不确定专业方向、升学就业或竞赛证书目标的学生。

【适用目标】
降低迷茫感，建立探索节奏。

【本月行动】
1. 列出自己感兴趣和不感兴趣的方向。
2. 各体验一次技术学习、社团活动和职业信息查询。
3. 找一位老师、学长学姐或辅导员聊一次。
4. 完成一个小任务，例如编程练习或简历初稿。
5. 月末选择下个月重点探索方向。

【回答边界】
这是探索建议，不要求学生立即确定长期目标。

【可信度】
C：成长建议。
